# Archive Docs

This directory stores historical planning and handoff documents.

Rules:
- Archived docs are not source-of-truth for current runtime behavior.
- Each archived file should start with `HISTORICAL` marker.
- Active operational and engineering guidance lives in:
  - `README.md`
  - `README_BACKEND.md`
  - `docs/runbooks/`
  - `AGENTS.md`
  - `CODE_STYLE.md`
  - `ENGINEERING_RULES.md`
  - `REPO_STRUCTURE.md`
