# Монетизаційна стратегія з енергією та міні-покупками

## 🎯 Ваша ідея: ГЕНІАЛЬНА! ⭐⭐⭐⭐⭐

**Чому це працює:**
- ✅ Звична механіка (як у мобільних іграх)
- ✅ Низький бар'єр входу (10⭐ = €0.12 — "не жаль спробувати")
- ✅ Варіативність (можна купити трохи або все)
- ✅ Імпульсні покупки (хочу зараз, не можу чекати!)
- ✅ Масовість (багато маленьких покупок = великий дохід)

**Аналогії, які працюють:**
- Candy Crush: життя закінчились → почекай або купи
- Duolingo: practice streak freeze за $1.99
- Mobile games: energy system + IAP (in-app purchases)

## 🔋 ЕНЕРГІЯ: Детальний дизайн системи

### Базова механіка

```
⚡ Енергія = гральна "валюта"

1 вікторина (5-7 питань) = -1 енергія

Початковий запас: 20 енергії
Відновлення: +1 енергія кожні 30 хвилин
Максимум: 20 енергії (не накопичується більше)

Повне відновлення: ~10 годин (з 0 до 20)
```

### Візуалізація для користувача

**У головному меню:**

```
🏠 Головне меню

⚡ Енергія: 15/20 [███████░░░]
Наступна +1 за 12 хв ⏰

[▶️ Грати] (потрібно 1⚡)
```

**Коли енергія закінчилась:**

```
😴 Енергія закінчилась!

⚡ 0/20

Відновиться повністю через 9 год 47 хв

Що робити?

[⏰ Почекати] 
[⚡ Купити енергію (10⭐)] 
[💎 Premium (безліміт!)]
```

### Виключення (що НЕ витрачає енергію)

```
Безкоштовно без енергії:

1. Daily Challenge (1 раз/день) — 0⚡
   Це щоденний "подарунок"
   
2. Турніри (якщо є активний Premium) — 0⚡
   Premium = безлімітні турніри

3. Challenge від друга (отримав виклик) — 0⚡
   Щоб не блокувати вірусність
```

**Чому це важливо:**
- Daily Challenge = щоденне повернення (retention)
- Турніри = value для Premium
- Челенджі = вірусність (не блокуємо!)

## ⚠️ Ризики та як їх мінімізувати

### Ризик 1: Користувачі не хочуть платити за енергію

**Проблема:** "Це як у мобільних іграх, не люблю!"

**Пом'якшення:**
- ✅ 20⚡/день = 20 вікторин (багато безкоштовно)
- ✅ Daily Challenge БЕЗ енергії (щоденний подарунок)
- ✅ Ціни низькі (10⭐ = €0.12, не €5 як у Candy Crush)
- ✅ Альтернатива: Premium безліміт (справедливо)

### Ризик 2: Занадто складна модель (3 рівні)

**Проблема:** Користувачі заплутаються

**Пом'якшення:**
- ✅ Чітка візуалізація (що дає кожна опція)
- ✅ Рекомендації ("BEST VALUE" на Month Pass)
- ✅ Просте пояснення: "Хочеш трохи → micro, хочеш все → Premium"

### Ризик 3: Канібалізація (micro їдять Premium)

**Проблема:** Всі купують micro, ніхто Premium

**Дані для відстеження:**
```
Micro buyers lifetime:
- Якщо купують micro >3 місяці → проблема
- Якщо переходять у Premium за 2-4 тижні → ОК

Цільова воронка:
Free → Micro (1-2 тижні) → Premium (тиждень 3-4)
```

**Пом'якшення:**
- ✅ Показувати value Premium частіше
- ✅ Обмежити micro покупки? (макс 3/день)
- ✅ "Витратив X⭐ на micro, краще Premium"

### Ризик 4: Користувачі чекають, не купують

**Проблема:** "Почекаю 10 годин, нащо платити"

**Пом'якшення:**
- ✅ Streak pressure ("не хочу втратити streak")
- ✅ FOMO ("limited time offer тільки зараз")
- ✅ Social pressure ("друзі вже грають далі")
- ✅ Турніри (потрібна енергія ЗАРАЗ для участі)
