# Fonts for Proof Card

Recommended fonts for the proof card renderer:

1. `Bebas Neue` from Google Fonts
2. `Montserrat` (Bold and optional Regular) from Google Fonts

Install steps:

1. Download files from https://fonts.google.com/specimen/Bebas+Neue and https://fonts.google.com/specimen/Montserrat
2. Put files into this folder with exact names:
   - `Bebas-Neue.ttf`
   - `Montserrat-Bold.ttf`
   - `Montserrat-Regular.ttf` (optional)

If these files are not present, renderer falls back to system fonts:
`DejaVuSans` / `LiberationSans` / `Arial` / `Calibri`.
